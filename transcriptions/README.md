# Transcription Output Files

This directory contains the saved transcription results from the YouTube Audio API.

## File Types

### ðŸ“ Text Files (`.txt`)
Plain text transcription of the audio content. Easy to read and edit.

**Format:**
```
[Full transcription text without timestamps]
```

### ðŸ“‹ JSON Files (`.json`)
Complete transcription data with timestamps and metadata.

**Format:**
```json
{
  "transcription": "Full text...",
  "sentences": [
    {
      "text": "Sentence text",
      "start": 0.0,
      "end": 3.5,
      "duration": 3.5
    }
  ],
  "metadata": {
    "file_size_mb": 112.1,
    "processing_time_seconds": 218.4,
    "processing_speed_mb_per_sec": 0.51,
    "chunks_processed": 9,
    "silence_removed": true
  }
}
```

### ðŸŽ¬ SRT Files (`.srt`)
Standard subtitle format compatible with video players and editing software.

**Format:**
```
1
00:00:00,000 --> 00:00:03,500
First sentence

2
00:00:03,500 --> 00:00:07,200
Second sentence
```

### ðŸ“Š Summary Files (`.txt`)
Quick overview of the transcription job with performance metrics.

**Contains:**
- Generation timestamp
- Performance metrics (download/transcription time)
- Content statistics (sentences, words, characters)
- List of generated files

## File Naming

All files use timestamp-based naming:

```
transcription_YYYYMMDD_HHMMSS.{txt,json,srt}
summary_YYYYMMDD_HHMMSS.txt
```

Example:
```
transcription_20260212_143022.txt
transcription_20260212_143022.json
transcription_20260212_143022.srt
summary_20260212_143022.txt
```

## Usage Examples

### Import JSON in Python

```python
import json

with open('transcription_20260212_143022.json', 'r') as f:
    data = json.load(f)
    
print(data['transcription'])
for sentence in data['sentences']:
    print(f"[{sentence['start']:.1f}s] {sentence['text']}")
```

### Use SRT with VLC/MPV

```bash
# Play video with transcription subtitles
vlc video.mp4 --sub-file transcription_20260212_143022.srt

# Or with mpv
mpv video.mp4 --sub-file=transcription_20260212_143022.srt
```

### Convert to Other Formats

```bash
# Convert SRT to VTT
ffmpeg -i transcription.srt transcription.vtt

# Convert to plain text (timestamps removed)
cat transcription.json | jq -r '.transcription' > transcription.txt
```

## Automatic Generation

Files are automatically saved when using:

- `python test_chunked_transcription.py [URL]` - Generates all formats
- `python example_transcription.py` - Generates all formats

## Tips

1. **Keep JSON files** - They contain the most complete data
2. **Use SRT for video editing** - Compatible with most software
3. **Share TXT files** - Easiest format for reading
4. **Archive old files** - Move completed projects to separate folders
